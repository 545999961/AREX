Overview

Agentic research with search and reasoning

AREX is designed as a general agent for information-seeking tasks that require more than a single lookup. It can search across sources, connect evidence, and reason through multi-step questions before producing an answer.

Capabilities

Built for long-horizon problem solving

Deep Search

Explores complex questions through iterative search, comparison, and evidence gathering.

Reasoning

Breaks down tasks, follows intermediate findings, and forms answers from connected context.

Agent Workflow

Coordinates search, reading, synthesis, and final response generation in a single agent loop.

Release

Released by BAAI

AREX is presented as a BAAI agent focused on deep search and reasoning capabilities for research-style tasks. The public announcement is available through BAAI.
