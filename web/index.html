Overview

Research agents that optimize information gain

AREX is designed for long-horizon tasks where a single retrieval step is not enough. Instead of relying only on larger inference budgets, AREX emphasizes efficient agent behavior: when to search, what to read, which evidence to keep, when to compress prior reasoning, and when to stop and synthesize.

AREX supports deep search, broad investigation, paper reading, and expert-style verification scenarios. Its goal is to make research agents both capable and deployable: strong reasoning and tool use with a smaller active compute footprint.

Capabilities

Built for long-horizon problem solving

Efficiency-First Search

Optimizes each interaction for useful information gain, reducing unnecessary browsing, inference cost, and response latency.

Agentic Context Management

Treats context as an active workspace, deciding when to preserve notes, compress old reasoning, and retrieve key evidence.

Robust Tool Use

Supports multi-round browsing, retrieval, data analysis, and verification workflows needed by complex open-domain questions.
